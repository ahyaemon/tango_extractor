# tango_extractor
## ローカルで動かすには
- このリポジトリをクローンするか zip でダウンロードして解凍する

- 依存するパッケージをインストール
  - リポジトリ直下で下記コマンドを実行
  ```
  yarn install
  ```

- 起動
  - リポジトリ直下で下記コマンドを実行
  ```
  yarn dev
  ```

## 使用したデータについて
- [はてなキーワード](http://developer.hatena.ne.jp/ja/documents/keyword/misc/catalog) からダウンロードさせていただいたデータを使用

## 他
- データベースにデータを入れるときに [hiragana] 列と [tango] 列を間違えて逆にしてしまったけど面倒だからそのまま
